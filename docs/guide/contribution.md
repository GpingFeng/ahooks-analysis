# 贡献指南

欢迎您的指正以及贡献。这篇指南会指导您如何为 ahooks analysis 贡献一份自己的力量，请在您要提 issue 或者 pull request 之前花几分钟来阅读一遍这篇指南。

## 分支管理

我们长期维护 main 分支。如果您要修一个 issue 或者提供一个新的 hook 源码解读，那么请发 pull request 到 main，我们会每两周进行一次发版。
